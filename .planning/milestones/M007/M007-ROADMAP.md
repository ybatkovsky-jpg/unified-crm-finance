# M007: Аналитика и дашборды

**Vision:** Сквозная аналитика: воронка продаж, маржинальность проектов, эффективность команды, метрики закупок. Единый дашборд с KPI.

## Slices

- [ ] **S01: Sales Funnel Analytics** `risk:medium`
  > API + UI воронки продаж: конверсия по стадиям, суммы, сравнение периодов

- [ ] **S02: Project P&L / Margin** `risk:medium` `depends:[M006/S03]`
  > Маржинальность проектов: P&L, margin%, топ проектов

- [ ] **S03: Team Performance** `risk:medium`
  > Метрики менеджеров: конверсия, суммы, активность

- [ ] **S04: Procurement Metrics** `risk:low` `depends:[M005]`
  > Сроки закупок, ТОП поставщиков, оборачиваемость

- [ ] **S05: Main Dashboard** `risk:medium` `depends:[S01,S02,S03,S04]`
  > Единый дашборд с KPI-карточками, фильтрами, виджетами
