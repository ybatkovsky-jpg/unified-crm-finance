## UAT S04: Счета от поставщиков

### Сценарии проверки

1. **Загрузка счета**
   - Ручная загрузка файла счета
   - Парсинг данных из файла

2. **Список счетов**
   - Отображение всех счетов с фильтрами
   - Статусы: draft, processing, matched, paid, rejected

3. **Сверка (reconciliation)**
   - Сопоставление позиций счета с заказом
   - Diff view для сравнения
   - Ручной matching

4. **Детализация счета**
   - Просмотр списка позиций
   - Связь с контрагентом и заказом
