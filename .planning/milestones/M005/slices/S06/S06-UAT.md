## UAT S06: Склад

### Сценарии проверки

1. **Список складских остатков**
   - Отображение всех товаров с остатками
   - Цветовая индикация: критично мало / мало / достаточно / много

2. **Детализация товара**
   - Просмотр истории транзакций
   - Поступления и списания

3. **Атомарность транзакций**
   - Проверка корректности остатков при параллельных операциях
   - Нет отрицательных остатков

4. **Интеграции**
   - Автоматическое обновление при приемке поставки
   - Автоматическое списание при отгрузке
